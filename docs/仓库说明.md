# 仓库说明

## 唯一主应用

本 Git 仓库根目录即为 **惠企政策大脑** 前端工程（原 `wqy-sidebar-copy`）。

请在仓库根目录执行：

```bash
npm install
npm run dev
```

默认开发地址：**http://localhost:8080**

勿再在子目录 `wqy-sidebar-copy/` 下开发（该目录已删除）。

## 已移除的重复副本

以下目录已从工作区删除，不再维护：

- `wqy/`、`wqy-sidebar/`（子模块副本）
- `integrated-policy-project/`（旧 policy-compass 嵌套）
- `red-policy-hub-main/`
- `policy-compass/`（根级子模块指针）
- 临时文件与 zip 包

若需历史代码，请从 Git 历史记录中恢复。
